# Fieldset
Styled fieldset component for modal windows.

## Usage

```jsx
<Fieldset
          items={[
                {
                  id: 1,
                  label: 'First item',
                  value: 'firstItem',
                },
                {
                  id: 2,
                  label: 'Second Item',
                  value: 'secondItem',
                },
                {
                  id: 3,
                  label: 'Third item',
                  value: 'thirdItems',
                },
              ]}
          icon="#account"
          onClick={()=>console.log('Fieldset')}
          title="Example"
        />
```
