### Accessible expandable list

#### List with less than 4 items
```jsx
<ExpandableList>
   <ExpandableList.Item>Apple</ExpandableList.Item>
   <ExpandableList.Item>Peach</ExpandableList.Item>
   <ExpandableList.Item>Orange</ExpandableList.Item>
   <ExpandableList.Item>Raspberries</ExpandableList.Item>
</ExpandableList>
``` 

#### List with more than 4 focusable items
```jsx
<ExpandableList>
   <ExpandableList.Item>
    <a href="https://en.wikipedia.org/wiki/Apple">
        Apple
    </a>
   </ExpandableList.Item>
   <ExpandableList.Item>
      <a href="https://en.wikipedia.org/wiki/Peach">
        Peach
      </a>
   </ExpandableList.Item>
   <ExpandableList.Item>
     <a href="https://en.wikipedia.org/wiki/Orange_(fruit)">
       Orange
     </a>
   </ExpandableList.Item>
   <ExpandableList.Item>
     <a href="https://en.wikipedia.org/wiki/Raspberries">
       Raspberries
     </a>
   </ExpandableList.Item>
   <ExpandableList.Item>
     <a href="https://en.wikipedia.org/wiki/Banana">
       Banana
     </a>
    </ExpandableList.Item>
   <ExpandableList.Item>
     <a href="https://en.wikipedia.org/wiki/Mango">
       Mango
     </a>
   </ExpandableList.Item>
</ExpandableList>
```
